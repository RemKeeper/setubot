# 小红书涩图搜索关键词参考

## 通用搜索格式
- 碧蓝航线涩图 → `%E7%A2%A7%E8%93%9D%E8%88%AA%E7%BA%BF%E6%B6%A9%E5%9B%BE`
- 终末地洛茜 → `%E7%BB%88%E6%9C%AB%E5%9C%B0%E6%B4%9B%E8%8C%9C`
- 洛茜 cosplay → `%E6%B4%9B%E8%8C%9C+cosplay`
- 洛茜 写真 → `%E6%B4%9B%E8%8C%9C+%E5%86%99%E7%9C%9F`
- 原神甘雨 → `%E5%8E%9F%E7%A5%9E%E7%94%98%E9%9B%A8`
- 崩坏雷电将军 → `%E5%B4%A9%E5%9D%8F%E9%9B%B7%E7%94%B5%E5%B0%86%E5%86%9B`

## 关键词组合技巧
- `角色名 + cosplay` → cosplay照片
- `角色名 + 写真` → 写真/同人图
- `游戏名 + 角色名` → 该角色相关
- `角色名 + 涩图/擦边` → 更直接但内容可能更少

## URL编码工具
Python: `urllib.parse.quote("碧蓝航线涩图")`
在线: https://www.urlencoder.org/

## 搜索结果过滤
搜索结果中视频帖占比很高（特别是cosplay类），建议：
1. 先用JS表达式获取帖子列表时就过滤掉 `hasVideo: true` 的帖子
2. 优先选择点赞数高的图片帖
3. 如果连续遇到视频帖，尝试换关键词或滚动加载更多
